# Основные комманды Git
---
|комманда | действие |
|---------|----------|
|**git add**  |*Добавляет содержимое рабочей директории*.|
|**git status**|*Показывает состояния файлов*.|
|**git commit**|*Берёт все данные, добавленные в индекс*.|
---
